# Video format conversion

纯浏览器端视频格式转换工具 — 所有处理在本地完成，文件不上传任何服务器。

## 功能特性

- **纯本地处理**：基于 ffmpeg.wasm，视频不离开浏览器，引擎缓存后可离线使用
- **App 适配预设**：内置抖音、微信、小红书、B 站等平台优化参数，一键配置
- **7 种输出格式**：MP4 (H.264)、WebM (VP9)、MOV、MKV、GIF、MP3、AAC
- **20+ 输入格式**：MP4、MOV、AVI、MKV、WebM、FLV、WMV、M4V、3GP、OGG、TS、VOB 等
- **批处理**：多文件同时转换，支持全选/反选、批量应用设置、ZIP 打包下载
- **国际化**：简体中文、繁體中文、English、日本語、한국어
- **双主题**：深色/浅色模式 × 青霓虹/粉霓虹配色
- **灵活参数**：画质 (CRF)、分辨率、帧率均可调节；音频格式自动切换为比特率控制

## 快速开始

1. 用浏览器打开 `index.html`
2. 拖放视频文件到页面区域（或点击选择 / Ctrl+V 粘贴）
3. 左侧面板选择输出格式或 App 预设
4. 调整画质、分辨率、帧率（可选）
5. 点击下载按钮保存结果

## 技术栈

- **ffmpeg.wasm** (`@ffmpeg/ffmpeg@0.12.10`) — 浏览器端视频转码引擎，首次加载约 31MB
- **JSZip** — 批量下载时打包 ZIP
- 零构建工具、零依赖安装，纯静态文件直接部署

## 浏览器兼容性

推荐使用 Chrome 或 Edge（Chromium 内核）。Firefox 和 Safari 性能较慢，但仍可使用。

## 隐私

所有文件处理均在浏览器本地完成，不会上传到任何服务器。详见[隐私政策](privacy.html)。

## 部署

将仓库文件部署到任意静态服务器即可（Nginx、GitHub Pages、Vercel 等），无需后端。
